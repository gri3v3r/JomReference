--Installation--
Install the component part before you install the plugin . Enable any additional features you may want on the native looking panel of the component. 

--Description--

It includes 6 features that can be enabled seperately and work together:

-Recommendation Permission (User needs permission to recommend)
-Business Mode (User can enter his career info and include business interaction in recommendation)
-Recommendation Withdrawal(User can delete the recommendation he gave)
-Ratings (Recommendation includes rating)
-User Manageable Recommendations (User can publish/unpublish a given recommendation)
-Recommendation Request (User can ask another user for a recommendation)
